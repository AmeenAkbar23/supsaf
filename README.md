# SUPSAF 🌍

SUPSAF is a simple platform for discovering beautiful and hidden places people usually never find online.

Instead of focusing on hotels, cafés, or bookings, SUPSAF is built for exploration — quiet beaches, hidden roads, waterfalls, forests, mountains, and peaceful places shared by real people from their own locations.

The goal is to create a community-driven platform where anyone can discover and share unique places from around the world.

## Features

* Discover beautiful places
* Explore hidden locations
* Community-driven platform
* Clean and minimal experience
* Mobile-friendly design
* Fast and lightweight interface

## Current Status

SUPSAF is currently in beta.

Right now we are:

* adding more places
* improving the experience
* building community features
* working on ratings and reviews
